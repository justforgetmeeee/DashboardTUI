# DashboardTUI

Консольная утилита для мониторинга системных ресурсов Linux в реальном времени. Написана на .NET 10 с использованием библиотеки Spectre.Console.

## Особенности
- Мониторинг загрузки CPU на основе обработки данных из `/proc/stat`.
- Отображение состояния оперативной памяти (используемая/общая) через `/proc/meminfo`.
- Вывод времени работы системы (Uptime).
- Интерактивный интерфейс с поддержкой горячих клавиш.
- Минимальное потребление ресурсов и отсутствие внешних системных зависимостей (кроме .NET Runtime).

## Технологический стек
- **Язык:** C# 14
- **Платформа:** .NET 10
- **Библиотеки:** Spectre.Console
